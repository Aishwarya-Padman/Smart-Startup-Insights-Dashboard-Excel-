# Smart Startup Insights Dashboard (Excel)

## 📊 Project Overview
This project is an Excel-based interactive dashboard created to analyze startup data such as funding, cities, and company distribution.

## 🔹 Features
- Data Cleaning and Structuring  
- Excel Formulas and Functions  
- Interactive Slicers  
- Data Visualization using Charts  
- Dashboard Creation  

## 📈 Insights
- Startup funding trends  
- City-wise distribution of startups  
- Investment patterns  

## 🛠 Tools Used
- Microsoft Excel  

## 🚀 Purpose
This project was created as part of practice to improve Excel, data analysis, and dashboard skills.
